​Manually Update Drivers on PC [Using Driver Easy: Free Version]

Published by - Admin

Driver Easy is a free driver updater tool for Windows that has access to millions of device drivers at the click of a button. The program is really easy to use, too. Because you can download drivers directly from the software, you don't have to worry about searching for them yourself.

Follow these 9 steps and update your PC drivers for free.

Step 1: Installing Driver Easy Software

Search For the Driver Easy Software on your browser.

Download the free version only.

Install the setup and launch the software.

Step 2: Scan the PC for Updates

Now open the software and click Scan Now.
The software will scan the PC for any updates that are required for your drivers.

Step 3: Downloading Available Updates

Select the driver you want to update, then press Update.

This will open a window "Create a system restore point". Choose the Manually Create option and then continue.

Now, wait till the driver is downloaded.

After downloading, install the driver manually by choosing the Manually Install option and then continue.

Step 4: Open File Location

Once you choose the Manually Install option, the file location where the updates are downloaded is opened.

As shown in the image, double click on the address bar to get the address of the file and then copy it.

Step 5: Open Device Manager

Open windows search bar using Win + S, this will open a search bar and then type "Device Manager".

After opening the "Device Manager" expand all the drivers by clicking the ">" symbol.

By expanding all the drivers mentioned there will help us to search for our required driver easily.

​Step 6: Searching the Driver to Update Manually

Right-click on the driver we downloaded and options will open. (Like shown in the image).

Now note down the "Current Driver name" (Here: Generic Software Component).

Search the current driver name i.e. Generic Software Component in Device manager.Press the first letter of your current driver name, here Press G repeatedly until you find the one you required.

​Step 7: Driver Found!

Now after finding the driver in the device manager.

Right-click the driver and select the "Update driver" option.

After that choose the "Browse my computer for drivers" option

Paste the address of the file we copied earlier inside the search box and press Next.

​Step 8: Installing the Driver

Now it will install the driver updates automatically.

You will get a "Windows has successfully updated your drivers" message.

​Step 9: Restart Your PC

After installing all the required drivers, restart your PC.
